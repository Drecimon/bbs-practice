### 获取验证码

| 方法   | API                           | 请求参数         |  返回参数               |       说明
|------ |------                          |------           | -------                | ------
|GET    | http://IP/api/verification     |                 |                        | 获取验证码
|       |                                | null            | String: verId          | 验证码id
|       |                                |                 | String: verImageBase64 | 验证码图片Base64
|POST   | http://IP/api/editorimage      | 2               |  2                     | 富文本图片上传地址
|       |                                | 见系统设计.md    | int: errno             | 上传文件名
|       |                                |                 | String[]: data         | 图片地址列表
